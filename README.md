# GNSS Seismic Monitoring Field Campaign

## Overview
This project focuses on improving seismic hazard models by addressing gaps in GNSS data coverage across central and southern California. A targeted field campaign was conducted to recover and resurvey GNSS sites that had not been updated in several years.

## Objectives
- Improve GNSS data coverage in central and southern California  
- Recover sites not measured in 5+ years  
- Support the Community Geodetic Model (CGM)  
- Improve earthquake preparedness and ground deformation monitoring  

## Methodology
- Reviewed GNSS datasets (SCEC, NGS, USGS, UCR)  
- Identified outdated or missing sites  
- Checked accessibility using Google Earth  
- Conducted field surveys (12–14 hour occupations)  
- Converted data to RINEX format  
- Processed and validated with NGS OPUS  

## Results
- 37 GNSS sites surveyed:
  - 25 Inland Empire  
  - 12 Parkfield area  
- Data quality confirmed using OPUS  
- Updated site documentation and field records  
- Improved dataset for seismic hazard analysis  

## Key Insights
- Field campaigns are critical for closing GNSS data gaps  
- Updated measurements improve earthquake response models  
- High-quality GNSS data strengthens geophysical research  

## Conclusion
This project successfully improved GNSS data coverage in California, supporting better seismic hazard models and contributing to research on crustal deformation and fault strain accumulation.

## Tools Used
- ArcGIS Pro  
- Google Earth  
- NGS OPUS  
- GNSS field equipment
